# 🌍 Global Data Skills Market Intelligence Dashboard

## 📌 Project Overview

This project presents a comprehensive analysis of the global data job market using Power BI and Excel. The dashboard provides insights into the most demanded technical skills, salary premiums, growth trends, and market stability across different countries and data-related job roles.

The goal is to help aspiring Data Analysts, Data Scientists, Data Engineers, and Business Analysts understand current market demand and make data-driven career decisions.

# 🎯 Business Problem

The data industry is evolving rapidly, making it difficult for students and professionals to identify:

- Which technical skills are most demanded globally.
- Which skills offer the highest salary premium.
- Which emerging technologies are growing fastest.
- Which skills provide long-term stability.
- How demand varies across countries and job roles.

This dashboard addresses these questions through data-driven analysis.

# 💡 Key Insights

### 🔹 Demand Insights

- SQL dominates global demand across selected markets.
- Python maintains strong demand across nearly all job categories.
- Tableau and Power BI remain highly relevant for analytics professionals.

### 🔹 Growth Insights

- Python shows the strongest growth acceleration.
- SQL continues to grow while maintaining high demand stability.
- Emerging cloud and big-data technologies continue expanding.

### 🔹 Salary Insights

- Spark commands the highest salary premium.
- AWS and Java also provide strong salary advantages.
- Excel and Power BI often appear below market median salary.

### 🔹 Stability Insights

- Python combines strong hiring demand with market stability.
- SQL remains one of the safest long-term technical skills.
- Tableau demonstrates balanced demand and consistency.
